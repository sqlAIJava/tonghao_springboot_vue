# 同好管理后台

## 环境要求
```
JDK >= 1.8 (推荐1.8版本)
Mysql >= 5.5.0 (推荐5.7版本)
Maven >= 3.0
Node >= 10
Vue >= 2.6.x
```

## 介绍
```
项目为SpringBoot为基础的后端，以及HTML5+VUE的前端；
运行项目前，你得
1：新建数据库tonghao，导入提供（tonghao-zz-sql-mysql）给你的表数据
2：后端，先Maven Clean ，再Maven Package 一下，注意数据库连接用户和密码，没问题直接启动
3：前端，你得确定你有能跑Vue前端的环境，然后按照tonghao-z-ui-vue中的README.md文档启动
```
